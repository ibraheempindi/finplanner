### Quick Summary

Version 1.0 introduces the first major release of Budget Planner with the following key improvements:

1. **Hide Plan Button** - Collapse the budget planning form while keeping the current plan visible
2. **Dynamic Category Management** - Add new budget categories without editing JSON
3. **Seamless Expense Category Creation** - Add categories directly from the expense recording form
4. **Enhanced Login** - Displays logged-in user email for better account visibility
5. **Improved UI Consistency** - Consistent button styling and layout throughout
